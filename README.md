# Contact Manager  mini CRM

**Краткое описание:**  
`Contact Manager  mini CRM` — это интуитивно понятное Android-приложение для управления контактами, взаимодействиями и аналитики . Это легковесная CRM-система, которая работает полностью офлайн, предоставляя полный контроль над клиентской базой прямо с вашего телефона.

---

## 📱 Основные возможности

### 📇 Управление контактами (CRM)
- **Полный цикл:** Создание, редактирование, удаление и просмотр контактов.
- **Детальная информация:** Имя, телефон, email, компания, должность и заметки.
- **Импорт:** Быстрый импорт контактов из телефонной книги устройства.
- **Фильтрация и поиск:**
    - Поиск по всем полям контакта.
    - Фильтрация по **статусу лида** (Новый, В работе, Переговоры, Конвертирован, Потерян).
    - Фильтрация по **категории лида** (Горячий, Тёплый, Холодный).

### 📞 Журнал взаимодействий
- **Фиксация активности:** Запись каждого звонка, встречи, сообщения (SMS, WhatsApp, Telegram, Email).
- **Оценка качества:** Визуальная шкала для оценки продуктивности контакта (от 0 до 100%).
- **Удобное управление:** Интуитивный свайп (Swipe-to-Delete/Edit) по записям журнала.

### 📅 События и планирование
- **Планирование:** Создание встреч, звонков и других событий, привязанных к контакту.
- **Умный статус:** Автоматическое отображение статуса события: «Просрочено», «Сейчас», «Через ... мин.».
- **Синхронизация:** Экспорт событий в Google Календарь одним нажатием (требуется разрешение).

### 📊 Дашборд и аналитика
- **Ключевые метрики:** Общее число контактов, разбивка по статусам и категориям.
- **Процент конверсии:** Автоматический расчет эффективности воронки продаж.
- **Статистика событий:** Количество встреч, звонков и других событий за период (сегодня, неделя, месяц).

## 🛠 Технологический стек

Проект демонстрирует использование современных технологий Android-разработки:

- **Язык:** [Kotlin](https://kotlinlang.org/)
- **Архитектура:** MVVM (Model-View-ViewModel)
- **База данных:** [Room](https://developer.android.com/training/data-storage/room) + [KSP](https://kotlinlang.org/docs/ksp-overview.html)
- **Асинхронность:** [Coroutines](https://kotlin.github.io/kotlinx.coroutines/) + [Flow](https://kotlinlang.org/docs/flow.html)
- **UI/UX:** Material Design 3, ViewBinding, Navigation Component, RecyclerView, Swipe-to-Delete
- **Интеграции:** ContentResolver (импорт контактов), Calendar Provider (синхронизация с Google-календарём)
